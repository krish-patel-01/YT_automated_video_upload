┌───────────────────────────────────────────────────────────────────────────┐
│ 🚀 HOW TO START                                                           │
└───────────────────────────────────────────────────────────────────────────┘

  1. Double-click: START_AUTOMATION.bat
  2. See message: "Automation is running. Press Ctrl+C to stop."
  3. Leave window open while uploading

┌───────────────────────────────────────────────────────────────────────────┐
│ 📁 WHERE TO PUT VIDEOS                                                    │
└───────────────────────────────────────────────────────────────────────────┘

  Drop videos here → C:\Users\ADMIN\Desktop\ToUpload\
  
  Supported formats: .mp4 .mov .avi .mkv .wmv .flv

┌───────────────────────────────────────────────────────────────────────────┐
│ ✏️ HOW TO ADD TITLE & DESCRIPTION (Optional)                              │
└───────────────────────────────────────────────────────────────────────────┘

  Create a text file next to your video:
  
  my_video.mp4              ← Your video
  my_video_metadata.txt     ← Create this in Notepad
  
  Inside the .txt file:
  
  TITLE: My Video Title
  
  DESCRIPTION: Video description here
  You can write multiple lines!
  
  TAGS: tag1, tag2, tag3
  
  PRIVACY: private
  
  CATEGORY: 22
  
  MADE_FOR_KIDS: no
  
  If you don't create this file → Uses video filename as title
  Missing fields → Uses defaults from config.json

┌───────────────────────────────────────────────────────────────────────────┐
│ 📊 CHECK UPLOAD STATUS                                                    │
└───────────────────────────────────────────────────────────────────────────┘

  ⏳ Still in ToUpload/       → Currently uploading
  ✅ In Uploaded/             → Successfully uploaded!
  ❌ In FailedUploads/        → Had a problem (check logs)
  
  View on YouTube → https://studio.youtube.com

┌───────────────────────────────────────────────────────────────────────────┐
│ ⚙️ SETTINGS                                                               │
└───────────────────────────────────────────────────────────────────────────┘

  Default Privacy: Private (only you can see)
  Default Category: People & Blogs
  Notify Subscribers: Yes
  
  To change → Edit config.json file

┌───────────────────────────────────────────────────────────────────────────┐
│ ❓ TROUBLESHOOTING                                                        │
└───────────────────────────────────────────────────────────────────────────┘

  Problem                     Solution
  ───────────────────────────────────────────────────────────────────────
  Video not uploading         Wait 10 seconds after copying file
  Wrong video info            Check _metadata.json file
  System won't start          Double-click START_AUTOMATION.bat
  Can't find video            Check YouTube Studio → Content
  
  Detailed logs → logs/youtube_automation.log
